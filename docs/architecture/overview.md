# Visão Geral da Arquitetura

O projeto é construído com Next.js 14 usando a App Router.

## Estrutura de Pastas

- `src/app`: Rotas da aplicação
- `src/components`: Componentes React
- `src/lib`: Utilitários e configurações
- `src/services`: Serviços externos
- `src/types`: Tipos TypeScript

## Módulos

- CRM: Gerenciamento de leads e pipeline de vendas
- Dashboard: Métricas e analytics
- Automação: Agendamento de postagens
- IA: Geração de conteúdo
