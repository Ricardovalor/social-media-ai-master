# Social Media AI Master Platform

Uma plataforma inteligente para gerenciamento de mídias sociais com gerador de conteúdo por IA.

## Funcionalidades

- Dashboard com métricas de engajamento
- Agendamento de posts
- Geração de conteúdo com IA
- Análises e gráficos de desempenho

## Tecnologias

- React
- TypeScript
- Tailwind CSS
- Vite

## Como executar

1. Instale as dependências: `npm install`
2. Execute em desenvolvimento: `npm run dev`
3. Build para produção: `npm run build`
